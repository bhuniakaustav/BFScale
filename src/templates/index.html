BANANA FOR SCALE

ULTIMATE MEASURING TOOL

ABOUT

Banana For Scale is an online measuring tool for measuring the dimensions of objects both large and small using reference objects. Banana For Scale was developed by master's students from UCD as a Final project for Software Engineering.

How To Use?

Upload your image using the upload button above. Select a reference object in the image and select which object is to be measured. Sit back and let BFS do it's job!
